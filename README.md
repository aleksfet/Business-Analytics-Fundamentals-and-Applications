# Module 12 Assignment: Business Analytics

## 📊 Overview
This project analyzes business performance data for a fictional company called **GreenGrocer**.  
The goal is to use business analytics techniques to understand sales, customers, and store performance.

The project includes **descriptive, diagnostic, predictive, and recommendation-based analysis**.

---

## 🧠 What This Project Does

- Generates realistic business data (sales, customers, stores)
- Analyzes total sales, profit, and performance
- Visualizes trends using graphs
- Examines customer segments and spending behavior
- Identifies relationships between business factors
- Builds a simple prediction model for sales
- Provides business recommendations
